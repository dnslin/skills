# UI Component Libraries Reference

Four component libraries unified under ui-search. Each provides an `llms.txt` catalog.

## Libraries

### Aceternity UI
- **URL**: https://ui.aceternity.com
- **llms.txt**: https://ui.aceternity.com/llms.txt
- **Components**: 103+ components, 120 Pro Blocks, 14 Templates
- **Categories**: Hero sections, text effects, backgrounds, cards, Pro Blocks, navbars, grids, modals, media
- **Highlights**: Production-ready hero sections, rich text animations, decorative backgrounds, premium Pro Blocks, full templates (SaaS, Agency, Portfolio)
- **Dependencies**: framer-motion, tailwindcss, clsx, tailwind-merge

### Magic UI
- **URL**: https://magicui.design
- **llms.txt**: https://magicui.design/llms.txt
- **Components**: 70+
- **Categories**: Animated text, buttons, device mockups, visual effects, backgrounds, layouts, data display
- **Highlights**: Landing page components, animated counters, device frames (iPhone/Android/Safari), marquee, dock, smooth cursor
- **Dependencies**: framer-motion, tailwindcss

### UI Layouts
- **URL**: https://www.ui-layouts.com
- **llms.txt**: https://www.ui-layouts.com/llms.txt
- **Components**: 59
- **Categories**: Forms/inputs, layouts/navigation, overlays/dialogs, animations, visual effects, 3D, media/galleries
- **Highlights**: Liquid glass, image masking, video masking, color picker, datetime picker, range slider, directional drawer
- **Dependencies**: framer-motion, tailwindcss, gsap (some), three.js (3D components), lenis (smooth scroll)

### ReactBits
- **URL**: https://reactbits.dev
- **llms.txt**: https://reactbits.dev/llms.txt
- **Components**: 139
- **Categories**: Text animations (22), Animations/cursors (33), Components (43), Backgrounds (41)
- **Highlights**: Largest collection — text animations, cursor effects, physics-based animations, shader backgrounds (liquid chrome, plasma, iridescence)
- **Dependencies**: Varies per component (framer-motion, three.js, gsap, ogl)
