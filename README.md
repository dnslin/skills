# Skills

Claude Code 技能集合，涵盖前端开发、设计、问题诊断、项目协作与开发者工具。

## 可用技能

- **agent-browser** - 自动化浏览器交互，用于测试、表单填写、截图和数据提取
- **caveman** - 使用极简压缩表达方式，减少 token 消耗并保持技术准确性
- **design-taste-frontend** - 前端设计指导与审美优化
- **diagnose** - 面向疑难 bug 和性能回退的诊断流程，覆盖复现、定位、修复与回归验证
- **find-skills** - 发现和定位可用技能
- **frontend-design** - 创建高质量的生产级前端界面
- **gh-create-issue** - 根据 PRD 或需求自动创建 GitHub Issue，支持复杂度评估与任务拆分
- **gpt-imagegen** - 使用 OpenAI 兼容 API 生成和编辑图片
- **grill-with-docs** - 基于文档的开发辅助
- **handoff** - 将当前对话压缩整理为可交接给下一位 agent 的 handoff 文档
- **improve-codebase-architecture** - 识别架构改进与重构机会，提升可测试性与 AI 可导航性
- **init-deep** - 初始化嵌套 CLAUDE.md 结构，实现渐进式上下文加载
- **prototype** - 构建一次性原型，用于验证设计、状态模型或 UI 方案
- **setup-matt-pocock-skills** - 为相关工程技能初始化项目上下文、Issue 跟踪器与文档约定
- **skill-creator** - 创建、评估和打包新技能
- **tdd** - 按 red-green-refactor 循环进行测试驱动开发
- **to-issues** - 将方案、规格或 PRD 拆分为可独立执行的 Issues
- **to-prd** - 将当前上下文整理为 PRD 并发布到项目 Issue 跟踪器
- **triage** - 按状态机和角色流程对 Issue 进行分诊与流转
- **ui-design-guided** - 引导式 UI 设计工作流
- **ui-search** - 搜索和引用 UI 组件库
- **vercel-react-best-practices** - Vercel 出品的 React 最佳实践与优化规则
- **web-design-guidelines** - Web 设计原则与规范
- **write-a-skill** - 创建具备规范结构和渐进式信息披露的新技能
- **zoom-out** - 从更高层视角梳理模块地图、上下文与整体结构

## 使用方式

每个技能是一个独立目录，包含定义其行为的 `SKILL.md` 文件。可将单个技能安装到 Claude Code 环境中，或作为构建自定义工作流的参考。
